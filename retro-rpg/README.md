# レトロ 8bit JRPG

懐かしの8bitスタイルJRPGです。HTML5 CanvasとVanilla JavaScriptで構築されています。

## 特徴
- **レトロなグラフィック**: 8bit風のドット絵とCRTモニター風のエフェクト
- **王道RPGシステム**: 経験値稼ぎ、レベルアップ、ゴールド集め
- **ターン制コマンドバトル**: 「たたかう」「じゅもん」「どうぐ」「にげる」のクラシックなスタイル
- **施設**: 宿屋で回復、武器屋で装備を強化

## 環境構築とインストール

このゲームをプレイするには、ローカルサーバーを起動する必要があります。以下のいずれかの方法で準備してください。

### 方法1: VSCode Live Server (推奨・一番簡単)
1. **VSCode** (Visual Studio Code) をインストールします。
2. VSCodeの拡張機能タブから **「Live Server」** を検索してインストールします。
3. VSCodeで `retro-rpg` フォルダを開きます。
4. `index.html` を右クリックし、**「Open with Live Server」** を選択します。
5. ブラウザが自動的に開き、ゲームが始まります。

### 方法2: Node.js / npm を使用する場合
コマンドラインに慣れている方はこちらが便利です。

1. **Node.jsのインストール**
   - [公式サイト](https://nodejs.org/) から推奨版（LTS）をダウンロードし、インストールしてください。
   - インストール完了後、ターミナル（コマンドプロンプトやPowerShell）で以下のコマンドを入力し、バージョンが表示されればOKです。
     ```bash
     node -v
     npm -v
     ```

2. **依存関係のインストール**
   `retro-rpg` フォルダで以下のコマンドを実行します。
   ```bash
   npm install
   ```
   ※ エラーが出る場合は、単に次の手順へ進んでください（このプロジェクトは依存関係がなくても動作可能です）。

3. **サーバーの起動**
   以下のコマンドで開発サーバーを立ち上げます（`vite`や`http-server`などが使われます）。
   ```bash
   npx http-server .
   # または
   npm run dev
   ```
   表示されたURL（例: `http://127.0.0.1:8080`）をブラウザで開いてください。

### 方法3: Pythonを使用する場合 (VSCode不要)
Pythonがインストール済みであれば、以下のコマンドだけでサーバーを起動できます。

1. コマンドプロンプトまたはPowerShellで `retro-rpg` フォルダを開きます。
2. 以下のコマンドを実行します。
   ```bash
   python -m http.server
   ```
3. `http://localhost:8000` をブラウザで開いてください。

### 操作方法
- **矢印キー**: 移動 / メニュー選択
- **Zキー**: 決定 / 会話 / 調べる
- **Xキー**: キャンセル / メニュー開閉

### ゲームの流れ
1. **城からスタート**: 王様の話を聞きましょう。
2. **フィールド探索**: 歩き回ってモンスターと戦い、レベルを上げましょう。
3. **町で買い物**: 武器屋で装備を整え、宿屋で回復しましょう。
4. **魔王討伐**: 強くなったら遠くへ冒険に出かけましょう（※プロトタイプ版のためマップは限定的です）。
