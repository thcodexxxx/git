# Shushi (家計簿・資産管理 Webアプリケーション)

![スクリーンショットのプレビュー (想定)](#)   <!-- 実際のスクリーンショット後で追加 -->

Googleスプレッドシートの手書き家計簿をベースにした、モダンで使いやすい収支・資産管理Webアプリケーションです。美しさと操作性を重視し、"Glassmorphism"（すりガラス効果）を取り入れたダークモダンなUIを採用しています。

## 主な機能 (Features)

1. **直感的なダッシュボード (Dashboard)**
   - 今月の収入、支出、収支差額、および現在の総資産額を一目で把握できます。

2. **月別収支管理 (Monthly View)**
   - 月ごとに収入と支出の履歴をリスト表示。
   - スプレッドシートと同様の感覚でカテゴリ別に明細を確認できます。

3. **総資産・口座残高管理 (Assets View)**
   - 楽天銀行、住信SBIネット銀行などのメインバンクに加え、NISA（積立・成長）やiDeCoなどの投資・運用資産の残高を一覧で管理します。

4. **クイック入力モーダル (Quick Add)**
   - ページ内のどこからでも「+ クイック追加」ボタンを押すことで、素早くその日の支出や収入を記録できます。

## 技術スタック (Tech Stack)

ビルド環境に依存せず、ブラウザだけで完結して動かせる軽量な構成です。

- **フロントエンド:** HTML5, Vanilla JavaScript
- **スタイリング:** Vanilla CSS (CSS Variables, Flexbox/Grid, Glassmorphism design)
- **アイコン:** Boxicons
- **フォント:** Google Fonts ('Inter')
- **データ保存:** Browser LocalStorage (サーバー不要でデータは端末内に保存されます)

## 使い方 (How to Setup & Use)

1. **リポジトリのダウンロード / クローン:**
   当ファイル群を含むフォルダ (`c:\git\shushi` などの作業フォルダ) を任意の場所に配置します。

2. **アプリの起動:**
   特別なインストールは必要ありません。いくつか方法があります。
   - **簡単:** フォルダ内の `index.html` を直接ブラウザ (Chrome, Edge, Safariなど) で開きます。
   - **ローカルサーバー利用 (推奨):**
     ターミナルを開き、該当フォルダ内で以下のPythonコマンドを実行すると、より安定して動きます。
     ```bash
     cd c:\git\shushi
     python -m http.server 8000
     ```
     その後、ブラウザで `http://localhost:8000` にアクセスします。

3. **テスト用データの注入:**
   右上の「テスト用モックデータを注入」ボタンをクリックすると、サンプルの収支・資産データ（スプレッドシート準拠）がLocalStorageに保存され、ダッシュボードの表示がどう変わるかをすぐに体験できます。

## 開発 (Development)

- **UI テーマ切り替え**: `style.css` 側の変数で制御しており、JavaScriptで `body` のクラスを切り替える構成です。デフォルトはダークモードです。
- **データ層**: `data.js` 内の `DataStore` オブジェクトがストレージ操作をラップしています。
